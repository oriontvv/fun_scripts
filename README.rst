Fun scripts
===========

|Build Status|

There are some interesting python tricks in this test playground repo.

Python3 required:

.. code:: bash

    $ python3.5 -m venv venv
    $ source venv/bin/activate
    $ pip install -r requirements.txt
    $ make tests
